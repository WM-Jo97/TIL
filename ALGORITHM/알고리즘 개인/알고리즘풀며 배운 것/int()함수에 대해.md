# int() 함수는
비교를 위해 반드시 사용해야한다!

```
A = input()
B = input()

if A > B :
    C=A+B
else :
    C=A*B
print(C)    
```

위 코드에서
    1
    2

를 입력받으면 어떤 결과를 얻을 수 있을까?

일반적으로 생각해보면
A가 작기 때문에 C = A*B 를 해서
C = 2 이다.

그러나 사실 컴퓨터가 인식하기로는 A와 B는 숫자가 아니라
문자이므로 곱하기를 할 수가 없다. 

그러므로 오류가 뜬다.

그럼 곱하기를 없애고 

```
A = input()
B = input()

if A > B :
    C=A+B
else :
    C=A
print(C)    
```
처럼 나타내면 어떤 결과가 나올까?

놀랍게도 C가 출력된다.

그러나 사실 숫자 A < B 이므로 
출력된 것은 아니고 A와 B를 문자로 인식해서
비교한 것이다.

따라서 반드시 int(A)와 int(B)로 바꿔주어야

원하는 연산이 가능하다
